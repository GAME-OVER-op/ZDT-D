2.2 Что записывать в список приложений?

В список приложений записывается имя пакета приложения. Имя пакета — это уникальный идентификатор приложения в системе Android, например, com.example.app.

Эти имена можно получить через команды в терминале, такие как:

  • pm list packages — для вывода всех установленных пакетов.
  • pm list packages | grep "название" — для поиска конкретного приложения.

Как работает список:

  • Если приложение указано в списке zapret, его трафик будет обрабатываться zapret.
  • Если приложение указано в списке DPI Tunnel, для него будут применяться правила iptables.

Добавление пакетов позволяет настроить точечную обработку, что полезно для ограничения или обхода блокировок трафика конкретных приложений.
